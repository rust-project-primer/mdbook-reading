# mdbook-reading



## Input

```
~~~article
link: "https://example.com/article.html"
title: "Article Title"
author: "John Doe"
archived: "/archive/john-doe-article.pdf"

Description of the article.
~~~
```

## Output

```
~~~admonish article title=""
Description of the article.
~~~
```
